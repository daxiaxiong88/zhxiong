# 我的摄影作品集（使用说明）

本项目是一个静态网页，展示个人摄影作品：不裁剪、不统一尺寸、无筛选。页面使用 Tailwind CSS（CDN 方式），并使用少量 JavaScript 仅用于“瀑布流布局”的列高平衡。

## 目录结构

- 胶片
- 数码

## 布局说明（Masonry）

- 页面为“瀑布流”布局：根据每张图片的真实高度自动分配到 1/2/3 列，使整体更均匀。
- 列数规则：手机 1 列（<640px）、平板 2 列（≥640px）、桌面 3 列（≥1024px）。
- 该分配通过少量 JavaScript 完成，不包含任何点击或筛选交互。


## 功能说明

- 按分类分区依次展示，点击放大。
- 保持图片原始比例与完整内容，不裁剪或统一尺寸。
